# 用語集

このドキュメントでは、プロジェクト内で使用される専門用語や略語を定義します。

| 用語             | 定義                                                                     |
| ---------------- | ------------------------------------------------------------------------ |
| フィード         | ユーザーのタイムラインに表示される投稿の集合                             |
| エンゲージメント | ユーザーがコンテンツに対して行う相互作用（いいね、コメント、シェアなど） |
| 無限スクロール   | ページの下部に到達すると自動的に新しいコンテンツを読み込む UI パターン   |
